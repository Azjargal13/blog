## Installation

``` sh
yarn add vuepress-theme-ououe
# or
npm i vuepress-theme-ououe
```

## Usage

``` js {3}
// .vuepress -> config.js
module.exports = {
  theme: 'ououe',
  themeConfig: {
    // ...
  }
}
```
